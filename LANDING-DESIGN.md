# Handoff: Toyla — ребрендинг сайта (казахский той-стиль)

## Overview
**Toyla (toyla.app)** — той-помощник: сервис создаёт пригласительные, собирает список гостей и автоматически рассылает им напоминания.

Задача хендоффа: **сменить визуальный стиль существующего сайта** на новое направление — *сдержанный казахский «той»-минимализм в духе хорошего Tilda-сайта* — и **добавить недостающие страницы** в этом же стиле.

Сейчас сайт оформлен в чужой теме (бирюзово-зелёный «ADM Home Care», шрифт Cormorant Garamond, e-commerce-вёрстка). Эту тему нужно **полностью заменить** на стиль, заданный референсом `reference/Toyla Landing.html`.

## About the Design Files
Файлы в папке `reference/` — это **дизайн-референсы, сделанные на HTML/CSS**. Это прототип внешнего вида и поведения, **а не продакшн-код для копипаста**.

Задача — **воссоздать этот стиль в существующем окружении сайта** (его текущем стеке/шаблонах), используя его установленные паттерны. Если выделенного фронтенд-окружения нет — реализовать на наиболее подходящем стеке. HTML-референс отражает намерение по цвету, типографике, отступам и композиции; перенесите систему, а не отдельные теги.

- `reference/Toyla Landing.html` — эталонная посадочная страница: хедер, герой, фото-блок, секция «3 шага», финальный CTA, футер. **Главный источник правды по стилю.**
- `reference/shanyrak.js` — генератор SVG-логотипа (шаңырак). Подключается и рисует знак в любой `[data-shanyrak]`-элемент. Перенести как компонент (React/Vue/web-component) или оставить как утилиту.
- `reference/image-slot.js` — заглушка-плейсхолдер для пользовательских фото (используется в герое). В реальном проекте заменить на обычный `<img>` / загрузчик.

## Fidelity
**High-fidelity (hifi).** Цвета, типографика, отступы и состояния — финальные. Воссоздавать пиксель-в-пиксель средствами своего стека. Значения — в разделах «Design Tokens» и «Components».

## Brand mark — Шаңырак (логотип)

Знак — **аутентичный шаңырак** (купол юрты сверху): обод (hoop) + изогнутая купольная решётка (күлдіреуіш) + радиальные спицы-уыки. Это национальный символ Казахстана; геометрия должна читаться именно как шаңырак, а не абстрактный круг.

Реализация — `reference/shanyrak.js`, функция `Shanyrak(opts)` возвращает SVG-строку (viewBox `0 0 100 100`):
- **spokes** (уыки): 28 радиальных линий, r 30.5 → 41, цвет `spoke`.
- **hoop** (обод): двойная окружность r=30 (`sw*1.1`) и r=27 (`sw*0.5`, opacity .5), цвет `hoop`.
- **lattice** (решётка): 3 вертикальных + 3 горизонтальных изогнутых пути (квадратичные кривые, прогиб ~4ед.), обрезаны по внутр. кругу r=27.5, цвет `lattice`.
- **knot**: центральная точка r=3.2.

Параметры: `{ hoop, lattice, spoke, knot, spokes=28, sw=2 }`.

**Стандартная окраска логотипа:**
- На светлом фоне: `hoop:#261B11`, `lattice:#A8492A` (глина), `spoke:#B0843A` (золото), `sw:2.3`.
- На тёмном фоне (ink): `hoop:#E9D9BE`, `lattice:#C8A24B`, `spoke:#C8A24B`.
- Как декоративный «водяной знак» (за текстом): однотонный (`#A8492A` или золото), `sw:1.4`, opacity 0.05–0.08.

**Логотип-локап:** знак (≈38px) + словесный знак `Toyla` шрифтом Spectral 500, 27px, цвет `#261B11`; финальная точка `.` цветом глины `#A8492A`.

## Design Tokens

### Цвета
| Токен | Hex | Назначение |
|---|---|---|
| `--bone` | `#F4ECDF` | Основной фон страницы (тёплый костяной) |
| `--bone-2` | `#EFE5D5` | Фон-вариант |
| `--paper` | `#FBF6EE` | Светлая поверхность / текст на тёмном |
| `--ink` | `#261B11` | Основной текст; тёмные блоки |
| `--ink-soft` | `#6E6151` | Вторичный текст |
| `--clay` | `#A8492A` | Акцент (терракота/глина): кнопки, акцентные слова, цифры |
| `--clay-deep` | `#8A3A20` | Hover акцента |
| `--gold` | `#B0843A` | Золото: eyebrow’ы, спицы лого |
| `--gold-soft` | `#C8A24B` | Золото на тёмном фоне |
| `--line` | `#E4D8C4` | Бордеры, разделители |

Палитра тёплая, сдержанная. **Не использовать** бирюзово-зелёный из старой темы, яркие градиенты, неоновые акценты. Один акцент (глина) + золото-нюанс.

### Типографика
- **Display / заголовки:** `"Spectral"` (Google Fonts), вес 300–600, **включая italic** (italic — для акцентных слов, цвет глины). Тёплый редакторский серив.
- **Текст / UI:** `"Manrope"`, вес 400–700.
- Подключение: `Spectral:ital,wght@0,300;0,400;0,500;0,600;1,400;1,500 & Manrope:wght@400;500;600;700`.

Шкала:
| Роль | Шрифт / вес | Размер | Прочее |
|---|---|---|---|
| H1 (hero) | Spectral 400 | `clamp(44px,6.4vw,78px)` | line-height 1.06; letter-spacing -.012em |
| H2 (секции/CTA) | Spectral 400 | `clamp(32px,4.4vw,56px)` | line-height ~1.1 |
| Заголовок секции | Spectral 400 | `clamp(32px,4vw,50px)` | |
| Eyebrow | Manrope 700 | 12px | uppercase, letter-spacing .30–.34em, цвет `--gold` |
| Lede / sub | Manrope 400 | 17px | line-height 1.6, цвет `--ink-soft` |
| Body | Manrope 400 | 15px | line-height 1.6 |
| Кнопка | Manrope 600 | 14.5px | |
| Цифры шагов | Spectral 300 | 46px | цвет `--clay` |

Акцентные слова в заголовках — `font-style:italic; color:var(--clay)` (на тёмном — `--gold-soft`).

### Радиусы / тени / отступы
- Радиусы: кнопки/чипы — pill `999px`; фото-фрейм — `24px`; тёмный CTA-блок — `32px`.
- Тень фото-фрейма: `0 30px 70px rgba(38,27,17,.14)`. Тень кнопки: `0 8px 20px rgba(168,73,42,.22)`.
- Контейнер: `max-width:1200px; padding:0 40px`.
- Вертикальные отступы секций: ~`104px`. Герой: `96px` сверху. Щедрый воздух — ключевая черта стиля.

### Компоненты
- **Кнопка primary:** фон `--clay`, текст `--paper`, pill, padding `12px 24px`, тень глины; hover → `--clay-deep`, `translateY(-1px)`. `white-space:nowrap`.
- **Кнопка light** (на тёмном): фон `--paper`, текст `--ink`.
- **Text-link:** Manrope 600 15px, нижний бордер 1.5px `--line`, hover → цвет/бордер `--clay`.
- **Хедер:** sticky, фон `rgba(244,236,223,.82)` + `backdrop-filter: blur(12px)`, нижний бордер. Высота 78px. Слева логотип, по центру nav, справа язык RU / «Кіру» / кнопка «Той құру».
- **Eyebrow:** золотой, uppercase, широкий трекинг (см. выше).
- **Фото-фрейм:** rounded 24px, бордер `--line`, большая мягкая тень; чип-подпись в углу (полупрозрачный paper, pill, мини-шаңырак).
- **Шаги (steps):** грид 3 колонки, разделены вертикальными бордерами `--line`, верхний бордер. Каждый: большая цифра (Spectral 300, clay) → KZ-заголовок (Spectral 500, 23px) → RU-лейбл (Manrope 700, 13px, uppercase, gold) → текст (Manrope, 15px, ink-soft).
- **Тёмный CTA-блок:** фон `--ink`, радиус 32px, золотой шаңырак-водяной знак справа (opacity .08), заголовок paper + italic gold.
- **Футер:** верхний бордер, в ряд: логотип · ссылки · копирайт.

## Двуязычность (важно для тона)
Стиль построен на паре **қазақша + русский**. Заголовки/кнопки/лейблы — на казахском, поясняющий текст — на русском. Сохранить этот приём:
- Кнопки: «Той құру» (создать той), «Кіру» (войти).
- Nav: «Мүмкіндіктер» (возможности), «Үлгілер» (шаблоны), «Баға» (цена), «Қолдау» (поддержка).
- Шаги: «Шақыру жасаңыз» / Приглашение; «Қонақтарды жинаңыз» / Гости; «Еске салдырыңыз» / Напоминания.
- Переключатель языка RU/KZ в хедере — функциональный.

## Brand assets (готовые файлы) — папка `brand-assets/`
Полный набор иконок и бейдж, уже в новом стиле (шаңырак, тёплая палитра). Открыть витрину: `brand-assets/Toyla — brand assets.html` (там же — готовый код для `<head>`).

**Favicon (вкладка браузера):**
- `favicon.svg` — векторный, основной (костяной фон, тёмный знак).
- `favicon-16.png`, `favicon-32.png`, `favicon-48.png`, `favicon-192.png`.

**App icon (iOS «на экран Домой» / PWA):**
- `apple-touch-icon.svg` + `apple-touch-icon-180.png` — ink-сквиркл, золотой шаңырак.
- `icon-192.png`, `icon-512.png` — для PWA-манифеста (maskable).

**Safari pinned tab:**
- `safari-pinned-tab.svg` — одноцветный (mono) знак, прозрачный фон; задаётся `color="#261B11"`.

**«Made with Toyla.» — бейдж для футера (например, на сайтах-приглашениях гостей):**
- `made-with-toyla-light.png` — для светлого фона.
- `made-with-toyla-dark.png` — для тёмного фона.
- Бейдж = мини-шаңырак + «Made with» (Manrope 500) + «Toyla.» (Spectral 500, точка — глина/золото). Можно собрать нативно по этим параметрам.

Код подключения (`<head>`):
```html
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon-180.png">
<link rel="mask-icon" href="/safari-pinned-tab.svg" color="#261B11">
```
> Примечание: знак детальный, поэтому в 16px favicon решётка слегка уплотняется — это норма; в 32px+ читается чисто. При желании можно сделать упрощённый 16px-вариант (меньше спиц).
